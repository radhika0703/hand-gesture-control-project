# 📷 Camera Project using OpenCV

This is a simple Python OpenCV project that opens the webcam and displays live camera feed in a window.

---

## 🚀 Features

- Access webcam using OpenCV
- Real-time video streaming
- Press `q` to exit the application

---

## 🛠️ Technologies Used

- Python
- OpenCV
